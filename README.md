# Tic-Tac-Toe AI

An implementation of an unbeatable Tic-Tac-Toe AI in Python using the Minimax algorithm.

The project was developed after learning about adversarial search in my Artificial Intelligence course. Rather than following an existing implementation, I designed and implemented the Minimax algorithm myself from the theoretical description to better understand recursive game-tree search and optimal decision making.

---

## Features

- Play against an unbeatable AI
- Recursive Minimax search algorithm
- Complete game-tree exploration
- Terminal state evaluation
- Simple graphical interface
- Written entirely in Python

---

## Motivation

One of the topics that interested me most during my AI course was adversarial search.

I wanted to move beyond understanding the theory and implement the algorithm myself from scratch. This project helped me understand how recursive search, state evaluation, and optimal decision making work together to create an AI that always selects the best possible move.

---


## Running the Project

Clone the repository

```bash
git clone https://github.com/nedasbartkus/tictactoe-minimax.git
cd tictactoe-minimax
```

Run

```bash
python game.py
```
